New benchmarks: July 2, 2009

The following is an update of the new benchmarks for 2009. It includes a number of fixes and repairs to the previously posted benchmarks as well as several new sets of benchmarks. The benchmarks previously posted as AUFLIRA have been recategorized as LRA (since they do not involve arrays, uninterpreted functions, or integers). Similarly, one set of benchmarks from AUFLIA has been moved to UFNIA. Note that only the first tarball for QF_BV has changed (i.e. QF_BV0). The other QF_BV tarballs are no different from the June 3 release.
